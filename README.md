# Resume-Scanner

## Project Overview

This project is a Python-based tool designed to parse Resume and Job Description (JD) PDF files, extract clean text, and prepare the data for an NLP matching algorithm (future implementation). The core functionality currently focuses on robust PDF text extraction and cleanup.
